# 流浪者与三个路人

一个 AI 叙事实验，验证一个问题：
**对话质量，能否在不依赖任何数值系统的情况下，
真正影响 NPC 在危机时刻的行为选择？**

---

## 核心机制

**Context Memory 替代 State Variables**——
传统好感度系统把对话压缩成数字。
本 Demo 用非结构化的上下文记忆替代结构化的状态机变量，
终局行为完全由大模型读取完整对话历史后自主判断。

**颜色系统作为关系深度的动态视觉化**——
黑色圆形代表一个被系统格式化的陌生人。
真诚的对话让颜色显现；套路或触发防线，颜色回黑。
双向可逆，实时反映当前对话质量。

**三个角色，三种心理原型，三种终局逻辑**——
压力反应类型（战/逃/僵）× 行动阈值（极高/极低/有条件）
决定危机时刻 NPC 的第一反应方向和被影响的方式。
相同的对话策略在不同角色身上产生截然不同的结果。
